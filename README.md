# Harsha Teja Bokkena - AI, ML & NLP Engineer Portfolio

Welcome to the source code for my professional portfolio! This repository houses the React application that showcases my background, certifications, and technical projects focusing on Artificial Intelligence, Machine Learning, and Natural Language Processing.

## 🚀 Technologies Used
- **Frontend Framework:** React (Vite builder)
- **Styling:** Modern utility classes
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Form Handling:** Web3Forms Contact Integration

## 🛠️ How to Run Locally

If you'd like to inspect my code or run the portfolio locally on your machine, simply follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/kannaharsha/PORTFOLIO.git
   cd PORTFOLIO
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Spin up the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
